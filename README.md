Description and Objective

Predict credit card default probability using payment history and demographic
data from a Taiwanese bank. The objective is to compare two ML methods
and estimate default probabilities.

Methods Used
Logistic Regression and Random Forest; report confusion matrix, ROC,
and probability estimates.
Dataset

Default of Credit Card Clients Dataset: 25,000 observations with payment
history and demographics. Similar Kaggle dataset: Default of Credit Card
Clients Dataset.
